# hello-world for react, vue and jinge

[English Readme](./README_en.md)

这个示例是最基本的 Hello World 示例。同时也包含 React 和 Vue 的 Hello World 项目的产物大小以兹对照。

三个项目都使用 `vite`，打包后的产物大小为：

| 框架  | 大小 | gzip 大小 |
| ----- | ---- | --------- |
| react | 140k | 45k       |
| vue   | 58k  | 23k       |
| jinge | 3.3k | 1.5k      |
